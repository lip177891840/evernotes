通过指定关键字来执行以前的命令
在下面的例子，输入 !ps 并回车，将执行以 ps 打头的命令：
# !ps
ps aux | grep yp
root 16947 0.0 0.1 36516 1264 ? Sl 13:10 0:00 ypbind
root 17503 0.0 0.0 4124 740 pts/0 S+ 19:19 0:00 grep yp
